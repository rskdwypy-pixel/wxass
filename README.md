# 微信公众号助手

## 安装说明

1. 创建图标文件（icon16.png, icon48.png, icon128.png）
   - 可以使用任何绿色主题的图标
   - 或者临时删除 manifest.json 中的 icons 配置

2. 在 Chrome/Edge 中加载插件：
   - 打开 chrome://extensions/
   - 开启"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择此文件夹

## 使用方法

1. 访问 mp.weixin.qq.com 并登录
2. 点击浏览器工具栏的插件图标
3. 点击"刷新认证信息"按钮
4. 在搜索框输入公众号名称并搜索
5. 点击搜索结果查看文章列表
6. 点击 fakeid 旁的"复制"按钮可复制 fakeid

## 功能特性

- ✅ 自动获取认证信息（Token 和 Cookie）
- ✅ 搜索公众号
- ✅ 显示公众号信息（名称、微信号、头像、fakeid）
- ✅ 一键复制 fakeid
- ✅ 查看公众号文章列表
- ✅ 复制文章链接
- ✅ 点击文章标题在新标签页打开
